# Prestige-Bias-and-Perceived-Credibility-of-Health-Information
Reproducible research report for a Prestige Bias study examining the effects of source prestige, need for cognition, and trust in science on perceived source credibility.
- This repository contains the published HTML version of my PSY 120L final reproducible report.
- This project examined whether source prestige influences perceived credibility of health information and whether this relationship is moderated by need for cognition and trust in science.
- The report was created using Quarto and R.
- The published report file is index.html.
- The dataset is not included in this repository.
